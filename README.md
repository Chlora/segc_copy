# SegC

## Instruções de compilação

**Requer Maven, JDK, e Java 17+**

Para compilar: mvn package
Para eliminar ficheiros: mvn clean

## Instruções de execução

#### Servidor

java -jar SpertaServer.jar [port]

#### Cliente

java -jar SpertaClient.jar [IP:port] [username] [password]
